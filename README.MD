# Подготовка проекта (Winodws)
```
git clone https://mrpchelka.visualstudio.com/CreditApp/_git/CreditApp
cd CreditApp
npm install
ionic cordova resources
XCopy xml resources\android\xml\
ionic cordova prepare
```
